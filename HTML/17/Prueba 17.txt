Basic HTML and HTML5: Create a Text Field

Now let's create a web form.

input elements are a convenient way to get input from your user.

You can create a text input like this:

<input type="text">

Note that input elements are self-closing.

Create an input element of type text below your lists.




HTML básico y HTML5: crear un campo de texto

Ahora creemos un formulario web.

Los elementos de entrada son una forma conveniente de obtener información de su usuario.

Puede crear una entrada de texto como esta:

<input type = "texto">

Tenga en cuenta que los elementos de entrada se cierran automáticamente.

Cree un elemento de entrada de tipo texto debajo de sus listas.

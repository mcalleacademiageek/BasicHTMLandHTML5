Basic CSS: Change the Font Size of an Element

Font size is controlled by the font-size CSS property, like this:

h1 {
  font-size: 30px;
}

Inside the same <style> tag that contains your red-text class, create an entry for p elements and set the font-size to 16 pixels (16px).


CSS básico: cambiar el tamaño de fuente de un elemento

El tamaño de fuente está controlado por la propiedad CSS de tamaño de fuente, como esta:

h1 {
     font-size: 30px;
}

Dentro de la misma etiqueta <style> que contiene su clase de texto rojo, cree una entrada para los elementos p y establezca el tamaño de fuente en 16 píxeles (16 px).

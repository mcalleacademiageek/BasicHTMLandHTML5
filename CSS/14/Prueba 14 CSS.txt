Basic CSS: Set the id of an Element

In addition to classes, each HTML element can also have an id attribute.

There are several benefits to using id attributes: You can use an id to style a single element and later you'll learn that you can use them to select and modify specific elements with JavaScript.

id attributes should be unique. Browsers won't enforce this, but it is a widely agreed upon best practice. So please don't give more than one element the same id attribute.

Here's an example of how you give your h2 element the id of cat-photo-app:

<h2 id="cat-photo-app">

Give your form element the id cat-photo-form.


CSS básico: establece la identificación de un elemento

Además de las clases, cada elemento HTML también puede tener un atributo id.

El uso de atributos de identificación tiene varios beneficios: puede usar una identificación para diseñar un solo elemento y luego aprenderá que puede usarlos para seleccionar y modificar elementos específicos con JavaScript.

Los atributos de identificación deben ser únicos. Los navegadores no harán cumplir esto, pero es una práctica recomendada ampliamente aceptada. Por lo tanto, no le dé a más de un elemento el mismo atributo de identificación.

Aquí hay un ejemplo de cómo le da a su elemento h2 la identificación de cat-photo-app:

<h2 id = "cat-photo-app">

Dale a tu elemento de formulario el id cat-photo-form.

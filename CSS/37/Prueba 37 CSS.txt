Basic CSS: Use CSS Variables to change several elements at once

CSS Variables are a powerful way to change many CSS style properties at once by changing only one value.

Follow the instructions below to see how changing just three values can change the styling of many elements.

In the penguin class, change the black value to gray, the gray value to white, and the yellow value to orange.


CSS básico: use variables CSS para cambiar varios elementos a la vez

Las variables CSS son una forma poderosa de cambiar muchas propiedades de estilo CSS a la vez cambiando solo un valor.

Siga las instrucciones a continuación para ver cómo cambiar solo tres valores puede cambiar el estilo de muchos elementos.

En la clase pingüino, cambie el valor de negro a gris, el valor de gris a blanco y el valor de amarillo a naranja.

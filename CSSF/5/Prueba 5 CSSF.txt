CSS Flexbox: Use the flex-direction Property to Make a Column

The last two challenges used the flex-direction property set to row. This property can also create a column by vertically stacking the children of a flex container.

Add the CSS property flex-direction to the #box-container element, and give it a value of column.


CSS Flexbox: use la propiedad flex-direction para hacer una columna

Los dos últimos desafíos utilizaron la propiedad flex-direction establecida en row. Esta propiedad también puede crear una columna apilando verticalmente los elementos secundarios de un contenedor flexible.

Agregue la propiedad CSS flex-direction al elemento # box-container y asígnele un valor de column.

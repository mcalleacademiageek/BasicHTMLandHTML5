CSS Flexbox: Apply the flex-direction Property to Create a Column in the Tweet Embed

The tweet embed header and footer used the flex-direction property earlier with a row value. Similarly, the items inside the .profile-name element would work well stacked as a column.

Add the CSS property flex-direction to the header's .profile-name element and set the value to column.


CSS Flexbox: aplique la propiedad flex-direction para crear una columna en el Tweet incrustado

El encabezado y el pie de página de inserción del tweet utilizaron la propiedad flex-direction anteriormente con un valor de fila. De manera similar, los elementos dentro del elemento .profile-name funcionarían bien apilados como una columna.

Agregue la propiedad CSS flex-direction al elemento .profile-name del encabezado y establezca el valor en column.

Applied Visual Design: Decrease the Opacity of an Element

The opacity property in CSS is used to adjust the opacity, or conversely, the transparency for an item.

    A value of 1 is opaque, which isn't transparent at all.
    A value of 0.5 is half see-through.
    A value of 0 is completely transparent.

The value given will apply to the entire element, whether that's an image with some transparency, or the foreground and background colors for a block of text.

Set the opacity of the anchor tags to 0.7 using links class to select them.


Diseño visual aplicado: disminuir la opacidad de un elemento

La propiedad de opacidad en CSS se utiliza para ajustar la opacidad o, a la inversa, la transparencia de un elemento.

     Un valor de 1 es opaco, que no es transparente en absoluto.
     Un valor de 0,5 es medio transparente.
     Un valor de 0 es completamente transparente.

El valor dado se aplicará a todo el elemento, ya sea una imagen con algo de transparencia o los colores de primer plano y fondo de un bloque de texto.

Establezca la opacidad de las etiquetas de anclaje en 0,7 usando la clase de enlaces para seleccionarlas.
